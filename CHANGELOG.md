# Changelog

## [0.1.0] - 2026-06-25

### Added

- Normalized run scoring with bounded quality and operational inputs.
- Explicit observable-trace-quality metric rather than a claim about hidden reasoning.
- Public-safe sample run artifact.
- v0.1 methodology, run-artifact schema, and public-release guidance.

### Scope

This is a starter benchmark structure for synthetic artifacts. It does not provide statistically meaningful leaderboard results, broad harness parity, repeated-run variance analysis, or provider endorsement.
